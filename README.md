# NicheHome.ch – Swiss By Velcheva Representative

Многостраничный статический сайт официального представителя бренда By Velcheva в Швейцарии. Проект повторяет архитектуру byvelcheva.es, но использует оригинальные тексты и заглушки, а также поддерживает четыре языка (DE/FR/IT/EN) на фронтенде.

## Языковая система

- Все тексты хранятся в объекте `translations` в `assets/js/script.js` (см. крупный комментарий `CHANGE YOUR REAL TEXTS HERE`).
- Каждому элементу в HTML назначен `data-i18n`-ключ, который совпадает со структурой объекта.
- Переключатель `DE | FR | IT | EN` в шапке обновляет тексты без перезагрузки страницы и сохраняет выбор в `localStorage`.
- Чтобы добавить язык:
  1. Скопируйте существующий объект (например, `en`).
  2. Измените ключи на новый код (`es`, `pt` и т. д.).
  3. Дополните переводами и добавьте кнопку в блок `.lang-switcher`.

## Работа с контентом

1. **Изменение текстов:** редактируйте соответствующие значения `translations[LANG].section.key`. После сохранения язык автоматически подтянет новое значение.
2. **Добавление карточек категорий/товаров:** продублируйте HTML-блок и создайте новые ключи в объекте перевода (например, `index.categories.card10`).
3. **Изображения:** замените SVG-заглушки в `assets/img/` своими фотографиями, сохранив пропорции.
4. **Форма:** фронтенд-вариант только валидирует обязательные поля и показывает успешное сообщение. Чтобы подключить реальную отправку, добавьте запрос в `initContactForm` (см. комментарий в JS).

## Стили и адаптив

- Сетка построена на CSS Grid/Flexbox, базовые переменные объявлены в `:root` (`--color-bg`, `--color-accent` и т. д.).
- Шапка фиксируется при скролле и сжимается благодаря классу `header--scrolled`.
- Мобильное меню (`header--open`) управляется бургер-кнопкой; навигация занимает весь экран на устройствах <1024px.
- Карточки и формы имеют скругления и лёгкие тени для премиального вида.

## Запуск проекта

```bash
# клонировать репозиторий
cd BV
python -m http.server 8000
# открыть http://localhost:8000 в браузере
```

Дополнительные серверы (Live Server, serve, netlify deploy) также подходят, так как проект полностью статический.

## Лицензия

Проект предоставлен «как есть» исключительно в демонстрационных целях и не содержит оригинальных материалов byvelcheva.es.
